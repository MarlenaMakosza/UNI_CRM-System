# CRM System Requirements

## Users
- 3 sales representatives
- Boss
- Simple UI — non-technical users

## Client database
- Adding companies (potential clients) from various sources
- Storing contact details of the decision maker

## Contacts & visits
- Logging phone calls
- Scheduling visits
- Orders can be placed during a phone call
- Post-visit notes: outcome, competitor prices
- Follow-up: call a few days after visit if no contract signed
- Follow-up: call the day after first delivery

## Contracts
- Recording signed contracts

## Manager reports
- Number of visits (per representative)
- Number and value of signed contracts
- Number of calls made
- Planned visits overview
- Turnover per client broken down by month
